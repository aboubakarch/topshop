# Node Version To Run TopShop

```
1. node >= 18
```

## Package Manager

- Yarn is used as a package manager in this project

## Script Added In this Project

- `yarn start`
- `yarn test`
- `yarn build`

### Steps to Run This Project

1. `First install node if not available`
2. `yarn install`
3. `yarn start`
4. `yarn test`
